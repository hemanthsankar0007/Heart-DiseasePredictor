### Medical Parameters:
The app analyzes 13 key medical indicators:
- Age, Sex, Chest Pain Type
- Blood Pressure, Cholesterol, Blood Sugar
- ECG Results, Heart Rate
- Exercise-Induced Angina
- ST Depression, Slope
- Major Vessels, Thalassemia
